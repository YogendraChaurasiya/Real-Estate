Appointment Count for Each Property

SQL Query Syntax:

        SELECT p.id AS property_id, p.address, COUNT(a.id) AS appointment_count
        FROM Property p
        LEFT JOIN Appointment a ON p.id = a.property_id
        GROUP BY p.id, p.address;

Output:

{% for property_id, address, appointment_count in appointment_counts %} {% endfor %}
